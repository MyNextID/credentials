# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Degree Certificate.

## Contents

- [edc-dc-example](edc-dc-example.json) - Example of an EDC-compliant Degree Certificate.
- [edc-dc-schema](edc-dc-schema.json) - JSON Schema definition of the EDC Degree Certificate.
- [edc-dc-signed](edc-dc-signed.jsonld) - Example of an EDC Degree Certificate signed with JAdES digital signature.
