# ELM

This folder contains resources related to the **European Learning Model (ELM)** credential format used for Degree Certificate.

## Contents

- [elm-dc-example](elm-dc-example.json) - Example of an ELM-compliant Degree Certificate.
- [elm-dc-schema](elm-dc-schema.json) - JSON Schema definition of the ELM Degree Certificate.
- [elm-dc-signed](elm-dc-signed.jsonld) - Example of an ELM Degree Certificate signed with JAdES digital signature.
