# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Degree Certificate.

## Contents

- [elm-dc-example](elm-dc-example.json) - Example of an EDC-compliant Degree Certificate.
- [elm-dc-schema](elm-dc-schema.json) - JSON Schema definition of the EDC Degree Certificate.
- [elm-dc-signed](elm-dc-signed.jsonld) - Example of an EDC Degree Certificate signed with JAdES digital signature.
