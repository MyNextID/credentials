# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Degree Certificate.

## Contents

- [schema](schema.json) - JSON Schema definition of the EDC Degree Certificate.
- [input-fields-to-credential-map](input-fields-to-credential-map.json) - Mapping between user-friendly input fields and the EDC Degree Certificate fields.
- [examples](examples) - Reference examples of an EDC-compliant Degree Certificate, including JAdES-signed instances.
