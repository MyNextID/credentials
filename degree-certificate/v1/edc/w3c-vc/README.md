# W3C Verifiable Credentials

This directory contains the **W3C Verifiable Credentials data model implementation** for the EDC Degree Certificate profile.
It defines the structure, validation rules, and mapping logic used to transform input fields into a W3C Verifiable Credential representation. It also includes example credentials for validation and reference.

## Contents

- [examples](examples) – Example credentials (unsigned and/or signed) used for validation and reference
- [input-fields-to-credential-map](input-fields-to-credential-map.json) – Mapping rules from input fields to credential attributes
- [schema.json](schema.json) – JSON Schema definition for the W3C Verifiable Credentials data model
