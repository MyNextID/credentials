# ISO/IEC 18013 Mobile Document (mDoc)

This folder contains resources related to the **ISO/IEC 18013 Mobile Document (mDoc)** format, used for the Personal ID.

## Contents

- [schema](schema.json) - JSON Schema definition of the ISO/IEC 18013 Mobile Document (mDoc) Personal ID credential.
- [input-fields-to-credential-map](input-fields-to-credential-map.json) - Mapping between user-friendly input fields and the ISO/IEC 18013 Mobile Document (mDoc) Personal ID fields.
