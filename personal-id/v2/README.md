# Personal ID (v2)

This folder defines the structure and resources for the v2 implementation of **Personal ID**.  

## Contents

- [input-fields](input-fields) - Translated input fields organised by language.
- [iso-iec-mdoc](iso-iec-mdoc) - ISO/IEC 18013 Mobile Document (mDoc) schemas and mappings.
- [sd-jwt-vc](sd-jwt-vc) - Selective Disclosure JSON Web Token - Verifiable Credential (SD-JWT-VC) schemas and mappings.
- [translations](translations) - Translated credential titles organised by language.
- [user-consent](user-consent) - Mapping between user consent groups and corresponding input fields.
