# SD-JWT

This folder contains resources related to the **Selective Disclosure JSON Web Token (SD-JWT)** format, used for the Personal ID.

## Contents

- [schema](schema.json) - JSON Schema definition of the SD-JWT Personal ID credential.
- [input-fields-to-credential-map](input-fields-to-credential-map.json) - Mapping between user-friendly input fields and the SD-JWT Personal ID fields.
