# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Matriculation.

## Contents

- [elm-matriculation-example](elm-matriculation-example.json) - Example of an EDC-compliant Matriculation.
- [elm-matriculation-schema](elm-matriculation-schema.json) - JSON Schema definition of the EDC Matriculation.
- [elm-matriculation-signed](elm-matriculation-signed.jsonld) - Example of an EDC Matriculation signed with JAdES digital signature.
