# ELM

This folder contains resources related to the **European Learning Model (ELM)** credential format used for Matriculation.

## Contents

- [elm-matriculation-example](elm-matriculation-example.json) - Example of an ELM-compliant Matriculation.
- [elm-matriculation-schema](elm-matriculation-schema.json) - JSON Schema definition of the ELM Matriculation.
- [elm-matriculation-signed](elm-matriculation-signed.jsonld) - Example of an ELM Matriculation signed with JAdES digital signature.
