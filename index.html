Credential Data Reference

This page provides access to auto-generated data for all supported credential types, including their translations, claims, and consent groups.

Available Data

Access the full dataset of all credential instances at credentials.json.

Access the list of all available credential types at credential-types.json.

Note: These files are automatically generated and located in the_autogen directory of the repository.

Source Repository

The original source and generation logic can be found in the credentials GitHub repository:

  • https://github.com/MyNextID/credentials

Advanced Usage

If you require more control over the data or only need a specific credential type, you can directly access the corresponding folder for that credential type in the repository.

Each credential type contains its own structure, including schema definitions, mappings, and examples. You can fetch and process this data according to your needs.

Credential Type Data

For a quick overview, you can retrieve data for a specific credential type as follows:

  • Find the desired credential type in credential-types.json .
  • Use the credential type value as the folder name: /<credential-type>/.
  • Select the desired version (e.g.v1or latest available).
  • Access the required resources within the version folder, such as:
    • Input fields
    • Input field translations
    • Mapping between input fields and credential format
    • User consent groups for input fields
    • Schemas and examples
