# Open Badge

This directory provides the repository’s implementation of the Open Badge profile for Microcredential, serving as a container for the profile’s supported data models, with links to their implementations.

## Contents – Available Data Models

- [w3c-vc](w3c-vc) - W3C Verifiable Credentials representation of the profile
