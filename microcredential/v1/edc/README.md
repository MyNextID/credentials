# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Microcredential.

## Contents

- [schema](schema.json) - JSON Schema definition of the EDC Microcredential.
- [input-fields-to-credential-map](input-fields-to-credential-map.json) - Mapping between user-friendly input fields and the EDC Microcredential fields.
- [examples](examples) - Reference examples of an EDC-compliant Microcredential, including JAdES-signed instances.
