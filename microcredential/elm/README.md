# ELM

This folder contains resources related to the **European Learning Model (ELM)** credential format used for microcredentials.

## Contents

- [elm-mc-example](elm-mc-example.json) - Example of a finalized ELM microcredential payload.
- [elm-mc-schema](elm-mc-schema.json) - JSON Schema definition for the ELM microcredential format.
- [elm-mc-signed](elm-mc-signed.jsonld) - Signed example of an ELM microcredential (JSON-LD).
- [input-fields-to-credential-map](input-fields-to-credential-map.json) - Mapping configuration between internal input fields and the ELM credential structure.
