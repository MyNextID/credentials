# ELM

This folder contains resources related to the **European Learning Model (ELM)** credential format used for Confirmation of Enrolment.

## Contents

- [elm-coe-example](elm-coe-example.json) - Example of an ELM-compliant Confirmation of Enrolment.
- [elm-coe-schema](elm-coe-schema.json) - JSON Schema definition of the ELM Confirmation of Enrolment.
- [elm-coe-signed](elm-coe-signed.jsonld) - Example of an ELM Confirmation of Enrolment signed with JAdES digital signature.
