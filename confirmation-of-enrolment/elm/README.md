# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Confirmation of Enrolment.

## Contents

- [elm-coe-example](elm-coe-example.json) - Example of an EDC-compliant Confirmation of Enrolment.
- [elm-coe-schema](elm-coe-schema.json) - JSON Schema definition of the EDC Confirmation of Enrolment.
- [elm-coe-signed](elm-coe-signed.jsonld) - Example of an EDC Confirmation of Enrolment signed with JAdES digital signature.
