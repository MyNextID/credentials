# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Confirmation of Enrolment.

## Contents

- [schema](schema.json) - JSON Schema definition of the EDC Confirmation of Enrolment.
- [input-fields-to-credential-map](input-fields-to-credential-map.json) - Mapping between user-friendly input fields and the EDC Confirmation of Enrolment fields.
- [examples](examples) - Reference examples of an EDC-compliant Confirmation of Enrolment, including JAdES-signed instances.
