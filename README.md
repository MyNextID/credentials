# Credentials

A centralized repository containing JSON schemas and examples for European Learning Model (ELM) and Open Badges 3.0 credential types.

## Overview

This repository provides standardized schema definitions and reference examples for digital credentials, supporting interoperability across learning and credentialing platforms. All examples are provided in unsigned format for development and testing purposes.

## Standards Compliance

- **ELM (European Learning Model)**: Aligned with EU standards for learning credentials
- **Open Badges 3.0**: Compatible with IMS Global Open Badges specification
