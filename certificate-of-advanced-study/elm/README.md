# ELM

This folder contains resources related to the **European Learning Model (ELM)** credential format used for Certificate of Advanced Study.

## Contents

- [elm-cas-example](elm-cas-example.json) - Example of an ELM-compliant Certificate of Advanced Study.
- [elm-cas-schema](elm-cas-schema.json) - JSON Schema definition of the ELM Certificate of Advanced Study.
- [elm-cas-signed](elm-cas-signed.jsonld) - Example of an ELM Certificate of Advanced Study signed with JAdES digital signature.
