
# ISO/IEC 18013-5 (ORG.ISO.18013.5.1)

This folder contains resources related to the **ISO/IEC 18013-5** standard for mobile driving licences (mDL), used for the **Mobile Driver's Licence** credential type.

## Contents

- [org.iso.18013.5.1-mdl-schema](org.iso.18013.5.1-mdl-schema.json) - JSON Schema definition of the ISO/IEC 18013-5 Mobile driver's licence.
- [schema](schema.json) - JSON Schema definition of the ISO/IEC 18013-5 Mobile driver's licence.
- [input-fields-to-credential-map](input-fields-to-credential-map.json) - Mapping between user-friendly input fields and ISO/IEC 18013-5 mDL data elements.
