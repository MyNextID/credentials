# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Certificate of Attendance.

## Contents

- [edc-coa-example](edc-coa-example.json) - Example of an EDC-compliant Certificate of Attendance.
- [edc-coa-schema](edc-coa-schema.json) - JSON Schema definition of the EDC Certificate of Attendance.
- [edc-coa-signed](edc-coa-signed.jsonld) - Example of an EDC Certificate of Attendance signed with JAdES digital signature.
