# ELM

This folder contains resources related to the **European Learning Model (ELM)** credential format used for Certificate of Attendance.

## Contents

- [elm-coa-example](elm-coa-example.json) - Example of an ELM-compliant Certificate of Attendance.
- [elm-coa-schema](elm-coa-schema.json) - JSON Schema definition of the ELM Certificate of Attendance.
- [elm-coa-signed](elm-coa-signed.jsonld) - Example of an ELM Certificate of Attendance signed with JAdES digital signature.
