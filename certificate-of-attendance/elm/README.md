# EDC

This folder contains resources related to the **European Digital Credentials (EDC)** format, a serialization of the ELM ontology, used for Certificate of Attendance.

## Contents

- [elm-coa-example](elm-coa-example.json) - Example of an EDC-compliant Certificate of Attendance.
- [elm-coa-schema](elm-coa-schema.json) - JSON Schema definition of the EDC Certificate of Attendance.
- [elm-coa-signed](elm-coa-signed.jsonld) - Example of an EDC Certificate of Attendance signed with JAdES digital signature.
